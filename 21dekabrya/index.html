Подари необычное хобби любителю крафтовых продуктов

Январь — отличное время для новых начинаний. Длинные выходные в начале месяца и температура за окном располагают к тому, чтобы остаться дома и открыть для себя какое-нибудь необычное хобби. Одним из таких хобби может стать самогоноварение и пивоварение.

Времена, когда это занятие ассоциировалось с какой-то подозрительной мутной жидкостью, прошли. Сейчас самогоноварение стало увлекательным хобби, которое покорило многих людей. Для некоторых оно стало чем-то наподобие соревнований, для других — шансом сделать уникальный качественный продукт своими руками.
